About the Application

This is a full-stack web application that helps users discover and interact with sports facilities across Hong Kong with interactive maps, event management, and facility reviews.

Features:

  • Interactive map to locate sports facilities
  • Event management for sports activities
  • User reviews and ratings for facilities
  • Authentication system for personalized experiences
